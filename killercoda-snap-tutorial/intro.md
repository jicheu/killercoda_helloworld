# Welcome to the Strictly Confined Snap Tutorial

In this interactive tutorial, you will learn how to:
1. Write a basic C++ application that writes an inspirational message to a file.
2. Package the application as an unconfined snap (`devmode`) to test it.
3. Enforce strict confinement and resolve permission issues by connecting interfaces.
4. Run the strictly confined snap inside an emulated Ubuntu Core image using LXD.

Let's get started!
